Semantic Server API to Linked Data Platform

What is?

The Semantic Server is an open-source REST API wrapper that for managing RDF entities stored as subject graphs. The current iteration uses Fedora 4 as a subject linked-data store and binary preservation store while providing expanded and enriched search of thses RDF entities through Elastic Search, fully supported by an HTTP SPARQL endpoint using Fuseki.
