Semantic Server API to Linked Data Platform

What is?

The Semantic Server is an open-source REST API wrapper that for managing RDF entities stored as subject graphs. The current iteration uses Fedora 4 as a subject linked-data store and binary preservation store while providing expanded and enriched search of these RDF entities through Elastic Search supported by an HTTP SPARQL endpoint using Fuseki.

Decoupling & Abstraction

The semantic server attempts to abstract the underlying search, preservation, and triple-stores functionality with a REST API for client applications.

The semantic server source code is available at https://github.com/jermnelson/semantic-server.git
